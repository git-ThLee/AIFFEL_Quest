# PRT(PeerReviewTemplate)

- [o] 코드가 정상적으로 동작하고 주어진 문제를 해결했나요?

  > 마크다운을 이용해 각 단계에 대한 해설을 작성해서 쉽게 이해할 수 있었다.
- [o] 코드가 에러를 유발할 가능성이 없나요?
  > 모델별, 데이터별로 변수를 다르게 설정해서 잘못된 모델, 데이터에 대해 처리되는 것을 방지하였다.
